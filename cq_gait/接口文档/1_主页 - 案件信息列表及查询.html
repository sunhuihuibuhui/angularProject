功能

  • 案件信息列表及查询-主页。

URL

{{portal_base}}case/caseinfo/list/

# portal_base: http://192.168.50.3:8006/

请求方式

  • GET

请求头

Authorization: JWT token

请求参数

  • 参数说明
字段 类型 描述
page string 哪一页
size string 每页多少条
jz_code string 案件编号
case_name string 案件名称
case_cate_gory string 案件类型
展示：来自案件类型树接口
请求：用户点击哪一个请求哪一个，多个逗号隔开
case_begin_time datetime 案发起始时间
case_end_time datetime 案发结束时间
label string 节点状态：
all:所有，
new:待提取，
pending:待审核，
ready_compare:待对比，
finished:已入库(弃用)
finished_compare:已对比，
finished_no_compare:不符合

响应数据格式

  • 响应体
字段 类型 描述
data dict 响应数据
data.succeed bool 是否成功
data.count int 总数
data.list list 当前页的数据对象列表
data.process_node_count dict 各节点状态的案件记录总条数
msg string 提示信息
code int 状态码

data.list

字段 类型 描述
visys_case_id(弃用) string 多维案件ID
visys\jz_code string 案件编号
visys_case_name string 案件名称
visys_case_cate_gory string 案件类型
visys_case_create_time string 案件创建时间(上传于时间)
visys_case_creator string 案件创建人(目前无，此处返回固定数据)
visys_case_context string 案件详情
visys_case_image_file_path string 案件代表图
case_cate_gory_path string 案件类型，父子层级关系，以逗号隔开
case_process_node string 节点状态，
new:待提取，
pending:待审核，
ready_compare:待对比
finished_compare:已对比，
finished_no_compare:不符合
other:未定义
verify_info list 审核人信息数组，排序：最新审核至最初审核
