功能

  • 比对详情 - 基本信息、原视频、步态特征、翻页信息。

URL

{{portal_base}}case/similar/detail/

# portal_base: http://192.168.50.3:8006/

请求方式

  • POST

请求头

Authorization: JWT token
Content-Type: application/json

请求体(过滤参数)

  • 参数说明
字段 类型 描述
page string 每次请求携带哪一页
size string 每次请求携带每页多少条
visys_case_id string 案件a的ID
visys_case_b_id string 案件b的ID
jz_code string 案件编号
case_name string 案件名称
case_cate_gory string 案件类型
展示：来自案件类型树接口
请求：用户点击哪一个请求哪一个，多个逗号隔开
case_begin_time datetime 案发起始时间
case_end_time datetime 案发结束时间
similarity_gte float 相似度大于等于
  • 示例
{
    "visys_case_id": "A50010862000020181100718",
    "visys_case_b_id": "A50010862000020181100737",
    "page": "1",
    "size": "20"
    // 以及，其他过滤参数
}

响应数据格式

  • 响应体
字段 类型 描述
data dict 响应数据
data.succeed bool 是否成功
data.data dict 详情数据
data.turn_pages dict 翻页信息
msg string 提示信息
code int 状态码

data.data

字段 类型 描述
visys_case_a_info dict 案件a的比对详情
visys_case_b_info dict 案件b的比对详情
similarity float 案件间的对比相似度
object_similarity_list list 目标间的对比相似度

visys_case_a_info 或 visys_case_b_info

字段 类型 描述
visys_case_id string 案件编号
visys_case_name string 案件名称
visys_case_cate_gory string 案件类型
visys_case_create_time string 案件创建时间(上传于时间)
visys_case_creator string 案件创建人(目前无，此处返回固定数据)
visys_case_context string 案件详情
visys_case_image_file_path string 案件代表图
case_cate_gory_path string 案件类型，父子层级关系，以逗号隔开
case_process_node string 节点状态，
all:所有，pending:未入库，ready_compare:未对比
finished_compare:已入库，finished_no_compare:已入库-不符合
other:其他，此状态页面目前不展示
case_videos list 案件的所有原视频数据
case_photos list 案件的所有目标数据

case_videos

字段 类型 描述
visys_file_type string 类型，解释：0:视频
visys_video_id string 视频ID
visys_video_file_path string 视频的URL路径
visys_video_create_time string 视频的创建时间
visys_video_name string 视频的名字

case_photos

字段 类型 描述
visys_file_type string 类型，解释：0:视频
visys_video_id string 视频ID
visys_video_file_path string 视频的URL路径
visys_video_create_time string 视频的创建时间
visys_video_name string 视频的名字
has_features list 该目标，在步态入库时，用户选择的步态特征(入库的)

case_features

字段 类型 描述
feature_id string 步态特征ID
feature_video_url string 步态特征视频-URL路径
feature_photo_url string 步态特征视频-封面截图URL路径

object_similarity_list

字段 类型 描述
visys_case_a_id string 案件a(右边)ID
visys_object_a_id string 案件a(左边)的目标ID
visys_case_b_id string 案件b(右边)ID
visys_object_b_id string 案件b(右边)的目标ID
similarity float 目标间的对比相似度
visys_object_a_name string 案件a(左边)的目标名称
visys_object_a_url string 案件a(左边)的目标图片url
visys_object_b_name string 案件b(右边)的目标名称
visys_object_b_url string 案件b(右边)的目标图片url

data.turn_pages

字段 类型 描述
pre_page_case_b_id string 上一页案件b的ID，没有则为空字符
next_page_case_b_id string 下一页案件b的ID，没有则为空字符
current_index string 当前案件b在所有案件b列表中，排第几
count string 案件b的列表中总条数
pre_page string 点击上一条，请求时携带的page
pre_size string 点击上一条，请求时携带的size
next_page string 点击下一条，请求时携带的page
next_size string 点击下一条，请求时携带的size
