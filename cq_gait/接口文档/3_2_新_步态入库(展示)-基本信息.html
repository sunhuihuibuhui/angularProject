功能

  • 步态入库 - 基本信息 - 案件信息及媒体文件 - 该案件所有步态特征的数据。

URL

{{portal_base}}case/gait/storage/basic/

# portal_base: http://192.168.50.3:8006/

请求方式

  • GET

请求头

Authorization: JWT token

请求参数

  • 参数说明
字段 类型 是否必传 描述
process_node string 是 节点的操作按钮，1:步态入库，0:重新步态入库
visys_case_id string 否(为空时，available_case必须存在且为1) 案件ID
available_case bool 否 标签的操作按钮，1:从头部标签点击进入的人工审核

响应数据格式

  • 响应体
字段 类型 描述
data dict 响应数据
data.succeed bool 是否成功
data.data dict 案件步态详情
msg string 提示信息
code int 状态码

data.data

字段 类型 描述
visys_case_id string 案件编号
visys_case_name string 案件名称
visys_case_cate_gory string 案件类型
visys_case_create_time string 案件创建时间(上传于时间)
visys_case_creator string 案件创建人(目前无，此处返回固定数据)
visys_case_context string 案件详情
visys_case_image_file_path string 案件代表图
case_cate_gory_path string 案件类型，父子层级关系，以逗号隔开
case_process_node string 节点状态，
all:所有，pending:未入库，ready_compare:未对比
finished_compare:已入库，finished_no_compare:已入库-不符合
other:其他，此状态页面目前不展示
case_medias dict 该案件相关的媒体文件
case_gait_features list 该案件的所有相关步态特征
pending_count int 未入库案件总数
operation string 在succeed为True时，有操作动作
1: 刷新本人工审核页面成功
0: 表示各种原因，需要跳转回主页

case_medias

字段 类型 描述
videos list 视频文件URL
photos list 图片文件URL

case_medias.videos 或 case_medias.photos

字段 类型 描述
visys_file_type string 类型，0:视频，1:图片
visys_video_id string 视频/图片ID
visys_video_file_path string 视频/图片的URL路径
visys_video_create_time string 视频/图片的创建时间
visys_video_name string 视频/图片的名字

case_gait_features

字段 类型 描述
visys_video_id string 视频ID
feature_id string 步态特征ID
feature_video_url string 步态特征视频-URL路径
feature_photo_url string 步态特征视频-封面截图URL路径
