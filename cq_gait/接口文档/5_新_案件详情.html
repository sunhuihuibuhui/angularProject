功能

  • 案件详情 - 任意节点。

URL

{{portal_base}}case/caseinfo/detail/{visys_case_id}/

# portal_base: http://192.168.50.3:8006/

请求方式

  • GET

请求头

Authorization: JWT token

url请求参数

  • 参数说明
字段 类型 描述
visys_case_id string 案件ID

响应数据格式

  • 响应体
字段 类型 描述
data dict 响应数据
data.succeed bool 是否成功
data.data dict 详情数据
msg string 提示信息
code int 状态码

data.data

字段 类型 描述
visys_case_id string 案件编号
visys_case_name string 案件名称
visys_case_cate_gory string 案件类型
visys_case_create_time string 案件创建时间(上传于时间)
visys_case_creator string 案件创建人(目前无，此处返回固定数据)
visys_case_context string 案件详情
visys_case_image_file_path string 案件代表图
case_cate_gory_path string 案件类型，父子层级关系，以逗号隔开
case_process_node string 节点状态，
all:所有，pending:未入库，ready_compare:未对比
finished_compare:已入库，finished_no_compare:已入库-不符合
other:其他，此状态页面目前不展示
case_medias dict 案件的媒体数据：videos(所有原视频)，photos(所有目标图片)
case_video_features list 案件的目标，每一个原视频，提取检测出来的所有步态序列
case_photo_features list 案件的目标，人工审核的每一个目标，所有确定嫌疑人的步态序列

case_medias.videos

字段 类型 描述
visys_video_id string 视频ID
visys_video_name string 视频的名字
visys_video_create_time string 视频的创建时间
visys_video_file_path string 视频的URL

case_medias.photos

字段 类型 描述
visys_video_id string 目标ID
visys_video_name string 目标的名字
visys_video_create_time string 目标的创建时间
visys_video_file_path string 目标的URL

case_video_features

字段 类型 描述
visys_video_id string 视频ID
visys_video_name string 视频的名字
visys_video_create_time string 视频的创建时间
visys_video_file_path string 视频的URL
has_features list 该原视频，提取检测出来的步态序列

case_video_features.has_features

字段 类型 描述
visys_video_id string 视频ID
feature_id string 步态特征ID
feature_video_url string 步态特征视频-URL
feature_photo_url string 步态特征视频-封面截图URL

case_photo_features

字段 类型 描述
visys_video_id string 目标ID
visys_video_name string 目标的名字
visys_video_create_time string 目标的创建时间
visys_video_file_path string 目标的URL
has_features list 该目标，人工审核入库的步态序列

case_photo_features.has_features

字段 类型 描述
visys_video_id string 目标ID
feature_id string 步态特征ID
feature_video_url string 步态特征视频-URL
feature_photo_url string 步态特征视频-封面截图URL
